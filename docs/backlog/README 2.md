# Backlog

This directory contains backlog and triage tracking for the Imagination Engine.

## How to use this

Agents and humans must append issues discovered outside their active work scope to `TRIAGE.md` rather than fixing them immediately.

## Structure

- `TRIAGE.md`: The active append-only log.

## Standards

Use the `[YYYY-MM-DD] Short description` format in the Open section of `TRIAGE.md`.
