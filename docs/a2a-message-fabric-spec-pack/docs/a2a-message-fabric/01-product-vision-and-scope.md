# 01 — Product Vision and Scope

The A2A Message Fabric lets agents, blocks, tools, workflows, canvas surfaces, model runtimes, OpenClaw execution cells, and observers exchange structured semantic events without duplicating the DAG engine or locking to one transport.

It enables DAG trace visibility, generative chaining, canvas block activity states, agent group orchestration, OpenClaw events, approval-gated actions, tool observability, artifact history, provenance hooks, device mesh/Edge Twin routing, model route visibility, and UI replay.

It is not a Mastra replacement, generic pub/sub toy, local-only EventEmitter, raw NDJSON pipe, prompt-injection channel, universal memory store, or permission bypass.
