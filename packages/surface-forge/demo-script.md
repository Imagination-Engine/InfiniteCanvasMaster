# Demo Script: Surface D - Forge (App Builder)

## Overview
This script outlines the exact sequence for Student D during the final presentation.

## Preparation
- Ensure Gemini 3.5 Pro API keys are active.
- Ensure the browser supports the WebContainers API natively.

## Execution Sequence

1. **Chat Shell:** Type: "Build a pomodoro timer app."
2. **Canvas Duality:** Observe the specific Forge agent blocks populate: `Architect`, `Designer`, `Builder`, `Tester`.
3. **Build Log Transparency:** Expand the Build Log UI. Watch the streaming thoughts as the agents iterate over the Blackboard state.
4. **Compilation:** Once the Tester passes, click 'Compile App'.
5. **Execution Sandbox:** The WebContainer boots within the browser. Interact with the Pomodoro timer directly on the canvas.
6. **Export:** Click "Launch as Creation" to open the app in a full-screen standalone view.
