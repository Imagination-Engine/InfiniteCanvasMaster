# Demo Script: Surface A - Playable (Game Studio)

## Overview
This script outlines the exact sequence for Student A during the final presentation.

## Preparation
- Ensure local server is running.
- Ensure DB is seeded with the generic "Platformer Base" components.

## Execution Sequence

1. **Sign Up / Login:** Open the web app and sign in as a new user.
2. **Chat Shell:** Type: "Create a simple platformer with a blue hero and basic physics."
3. **Canvas Duality:** Wait for the canvas to materialize. Observe the generated nodes: Scene, Character, Rule (Gravity), Rule (Collision).
4. **Live Edit:** Click the 'Playable Mode' toggle. Use the keyboard to move the character. 
5. **Multiplayer Sync:** Open a second incognito window, navigate to the invite link. Move the character in window 1, observe real-time movement in window 2.
6. **Export:** Click "Launch as Creation" to show the isolated `/play` route.
